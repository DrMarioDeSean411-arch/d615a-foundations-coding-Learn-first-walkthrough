# D615A · Foundations of Coding — Learn

A single-file, dependency-free HTML learning tool for WGU D615A. Unlike the
practice-exam tools in this suite, this one is **learn-first**: it teaches
each competency with explanations and worked examples before testing you,
rather than jumping straight to timed questions.

**Live:** `https://drmariodesean411-arch.github.io/d615a-foundations-coding-Learn-first-walkthrough/d615a-learn.html


## What's inside

- **C1 — Algorithmic Thinking & Programming** *(Python)*: variables/types,
  control structures, functions, lists & iteration
- **C2 — Troubleshooting & Debugging** *(Python)*: error categories,
  a systematic debugging process, reading tracebacks & print debugging,
  common bug patterns
- **C3 — Relational Databases & SQL** *(SQL)*: the relational model,
  `SELECT` filtering/sorting/aggregating, `JOIN`s, modifying data &
  constraints
Code examples, exercises, and knowledge checks in C1/C2 use Python
throughout — matching the zyBooks/course materials for D615A, which
teach programming fundamentals in Python and reserve a separate language
only for the database module (SQL, in C3). A few Python-specific
teaching points are called out directly since they don't have a JS/Java
equivalent: traceback read order ("most recent call last"), `None` vs.
falsy-value checks, mutable default arguments, and `IndentationError` as
a distinct syntax-error subtype.
 
Each of the 12 topics has: teaching prose → a worked example → an
interactive paste-and-compare exercise → 5 knowledge-check questions.
That's 60 questions total, answer order shuffled per load. There's also a
**Mixed Review** mode that draws 15 random questions across all three
competencies for spaced-practice-style review, independent of the taught
order.
