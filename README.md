# D615A · Foundations of Coding — Learn

A single-file, dependency-free HTML learning tool for WGU D615A. Unlike the
practice-exam tools in this suite, this one is **learn-first**: it teaches
each competency with explanations and worked examples before testing you,
rather than jumping straight to timed questions.

**Live:** `https://drmariodesean411-arch.github.io/<repo-name>/d615a-learn.html`
*(swap `<repo-name>` for whichever repo you drop this into — see below)*

## What's inside

- **C1 — Algorithmic Thinking & Programming**: variables/types, control
  structures, functions, arrays & iteration
- **C2 — Troubleshooting & Debugging**: error categories, a systematic
  debugging process, stack traces & print debugging, common bug patterns
- **C3 — Relational Databases & SQL**: the relational model, `SELECT`
  filtering/sorting/aggregating, `JOIN`s, modifying data & constraints

Each of the 12 topics has: teaching prose → a worked example → an
interactive paste-and-compare exercise → 5 knowledge-check questions.
That's 60 questions total, answer order shuffled per load. There's also a
**Mixed Review** mode that draws 15 random questions across all three
competencies for spaced-practice-style review, independent of the taught
order.

Everything runs client-side. No build step, no CDN calls, no
`localStorage` — progress lives in memory for the length of the browser
tab, same as the rest of the suite.
